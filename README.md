# NATU AI ğŸŒ

NATU adalah sistem berbasis **Artificial Intelligence (AI)** yang dikembangkan menggunakan pendekatan **Machine Learning (ML)** untuk menganalisis dan memprediksi potensi bencana alam secara **seakurat mungkin**.

Model-model pada NATU dirancang khusus untuk mengolah data lingkungan dan cuaca (BMKG) guna mendukung **mitigasi bencana dini**.

NATU berfokus pada pengolahan data terstruktur (**CSV/JSON**) seperti curah hujan, durasi hujan, tutupan awan, intensitas hujan, riwayat banjir, serta faktor geografis lainnya. Data tersebut diproses menggunakan model ML terlatih untuk menghasilkan **prediksi probabilitas bencana** dan **status risiko**.
